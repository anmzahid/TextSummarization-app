# Conversational Text Summarizer

This is a web application built with React that takes conversational text as input and generates summaries using BERT and PEGASUS models.

## Features
- Summarizes lengthy conversational texts.
- Uses BERT and PEGASUS models for text summarization.
- User-friendly web interface.
  
## Usage
1. Open the web application in your browser.
2. Enter the conversational text you want to summarize.
3. Click on the "Summarize" button.
4. View the generated summaries using BERT and PEGASUS models.

## Dependencies
- [React](https://reactjs.org/)
- [axios](https://github.com/axios/axios) (for making HTTP requests)
- [React Router](https://reactrouter.com/) (for navigation)
